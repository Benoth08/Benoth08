*Personal R&D · Public vitrine on GitHub*

Local conversational assistant with a neuro-inspired multi-layer
architecture: episodic + semantic memory (MHN + SDM + ChromaDB +

### EXALIA — Hybrid Liquid Neural Architecture
*Experimental · Advanced prototype*

Experimental architecture combining Large Language Models (LLMs) with
continuous-time neural networks (Liquid Networks) for advanced causal
reasoning.

**Engineering approach:**
- **Architecture** — Integration of Closed-form Continuous-time (CfC)
  networks within frozen Transformer backbones.
- **Mechanism** — Sparse routing directing static knowledge to
  pre-trained layers and dynamic states to liquid neural networks.
- **Hypothesis** — Address the "static logic" limitation of LLMs by
  introducing temporal dynamics for multi-step reasoning.

**Stack:** `PyTorch` `Hugging Face (Qwen)` `Neural Circuit Policies`
`torchdiffeq`
