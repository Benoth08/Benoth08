## Research & Innovation

### **Project EXALIA: Hybrid Neural Architecture**

Development of an experimental architecture combining Large Language Models with continuous-time neural networks for advanced reasoning capabilities.

**Technical approach:**
- Integration of Closed-form Continuous-time (CfC) networks within frozen Transformer backbones (Qwen)
- Sparse routing mechanism directing static knowledge to pre-trained layers and dynamic states to liquid neural networks
- Top-down feedback loop enabling iterative reasoning and multi-step logical inference
- Fusion layer combining symbolic knowledge with temporal dynamics

**Objective:** Enable LLMs to process sequential logic and causality as continuous temporal dynamics, supporting complex multi-step reasoning tasks.

**Technical stack:** PyTorch, Hugging Face Transformers, torchdiffeq, Neural Circuit Policies

**Key innovation:** The feedback mechanism allows the model to refine predictions iteratively, successfully addressing multi-step logical problems where standard LLMs typically fail.
