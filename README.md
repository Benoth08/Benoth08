## 🧠 Innovation & R&D : Project Taelys

**Initiative personnelle : "La chirurgie des LLM"**

Je développe **Taelys**, une architecture expérimentale visant à combler le fossé entre l'IA générative et la physique du monde réel :

* **Concept :** Hybridation de modèles Transformers (Qwen) avec des **Réseaux Neuronaux à Temps Continu** (Neural ODEs / CfC).
* **Objectif :** Permettre aux LLM de "comprendre" la dynamique temporelle continue et non-régulière.
* **Tech Stack :** PyTorch, Hugging Face, Sparse Upcycling, Liquid Networks.
